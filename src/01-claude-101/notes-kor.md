
감동 포인트
- 클로드는 인간의 불법적이거나, 비윤리적인 활동에 가담하는 것을 돕지 않는다.

클로드가 잘하는 것
1. 글쓰기 및 콘텐츠 제작
2. 연구 및 분석
3. 코딩 지원
4. 문제 해결 및 추론
5. 새로운 학습

클로드 접근성
1. claude.ai : 챗봇 (chatGPT)
2. claude code : 
  - 개발자를 위해 설계된 에이전트 기반 코딩 도구
  - 데스크톱의 모든 종류의 파일 제어가 가능
3. claude + slack
4. claude + excel

답변을 잘 받는 방법 (in claude.ai)
1. 상황 설정 : 내 배경과 목표에 대한 설명
2. 과제 정의 : 어떠한 행동을 시키고 싶은건지에 대한 설명
3. 규칙 명시 
4. 맥락 추가 : 기존 문서, 이미지와 같은 자료를 함께 첨부
5. 모델 선택 : 내 질문에 적합한 모델을 선택 (비용과도 관련이 있을지도...)
6. 추가 동작 :
  - 확장 사고 모드 on/off
  - 리서치 모드 on/off

AI와의 상호작용을 위한, 4D 프레임워크
1. 위임 : 어떤 일은 사람이, 어떤 일은 AI가 를 분배하는 것
2. 설명 : AI 시스템과 효과적으로 소통하는 것(출력 결과 명확히 정의, 프로세스 안내 등)
3. 분별력 : AI의 결과를 신중하고, 비판적으로 평가하는 능력
4. 성실성 : AI를 책임감 있고 윤리적으로 사용하는 것

RAG(Retrieval Augmented Generation) at Claude.ai
- 프로젝트에 지식 기반(참고문서, 배경자료, 예시, 기술문서, 사양 등)을 추가하여 AI가 더 정확하고, 관련성 높은 답변을 생성하도록 돕는 기술

## **방법 1: PowerShell 자동화 스크립트 (가장 권장) ⭐**

Windows 11에서 스크린샷을 바로 자동으로 저장하고 경로를 출력하는 PowerShell 스크립트:

```powershell
# screenshot-to-clipboard.ps1
$timestamp = Get-Date -Format "yyyyMMdd_HHmmss"
$filepath = "$env:TEMP\screenshot_$timestamp.png"

# 스크린샷 찍기 (Windows 11 스니핑 도구)
$wshell = New-Object -ComObject WScript.Shell
$wshell.SendKeys('%{PRTSC}')  # Alt+PrintScreen (활성창만)
Start-Sleep -Milliseconds 500

# 클립보드의 이미지를 PNG로 저장
$img = [Windows.Forms.Clipboard]::GetImage()
if ($img) {
    $img.Save($filepath)
    Write-Host "Screenshot saved: $filepath"
} else {
    Write-Host "No image in clipboard"
}
```

**사용법:**
```powershell
PowerShell -ExecutionPolicy Bypass -File "C:\path\to\screenshot-to-clipboard.ps1"
```
